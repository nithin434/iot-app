# Component Documentation

This folder contains component images and diagrams for tutorials and projects.

## File Naming Convention
- `components.png` - Main components overview
- `arduino-uno.png` - Arduino UNO pinout and specs
- `esp32.png` - ESP32 pinout and specs
- `dht22-sensor.png` - DHT22 temperature/humidity sensor
- `servo-motor.png` - Servo motor wiring diagram
- `lcd-display.png` - LCD display connection diagram
- `relay-module.png` - Relay module wiring
- `led-rgb.png` - RGB LED color codes and pinout

## Usage
Reference these images in tutorial content to help users understand:
1. Component connections
2. Pinout diagrams
3. Wiring schematics
4. Color codes and specifications
